# 1. Introduction and Goals

## Overview

GitLab AI Review Bridge is a lightweight orchestration layer between GitLab Merge Request discussions and AI coding agents such as Claude Code, Codex Agent, or VSCode-integrated agents.

Primary objective:

Convert GitLab review discussions into structured AI-ready execution tasks.

## Business Goals

- Reduce review friction
- Preserve review context
- Reduce context switching
- Prevent missed comments
- Standardize AI prompts

## Non-Goals

- Autonomous coding loops
- Semantic merge conflict resolution
- AI-generated commits
- Enterprise orchestration
